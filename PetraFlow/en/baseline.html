How to Use This Page#

The baseline page is not a replacement for full benchmark reports. It is meant to show:

  • Which modeling routes have already been compared against external references or direct JuMP models.

  • Which test files can serve as regression templates for new formulations and components.

  • Which results are intentionally summarized while raw logs remain private.
