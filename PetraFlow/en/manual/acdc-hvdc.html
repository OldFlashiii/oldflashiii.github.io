AC/DC and HVDC#

PetraFlow treats AC-only networks as the default. AC/DC hybrid modeling is enabled explicitly during initialization.

Load PGLIB-HVDC Data#

case_path = joinpath(PETRAFLOW_DATA_DIR, "PGLIB-HVDC", "case5_3_he.m")
power = load_power_network(case_path)
hvdc_case = load_pglib_hvdc_case(case_path)

HVDCMatACDCCase stores the original AC bus, AC branch, generator, cost, DC bus, converter, and DC branch matrices.

Initialize a Hybrid Context#

ctx = power_context(power; T = 1)

init_power!(
    ctx;
    formulation = :acp,
    state_variables = :polar,
    network_type = :acdc,
    hvdc_case = hvdc_case,
)

Accepted network-type aliases:

Value

Meaning

:ac, :default, :pure_ac

AC-only model

:acdc, :hybrid, :mixed_acdc

Hybrid AC/DC model

true

Alias for :acdc

false

Alias for :ac

add_modules!(
    ctx,
    ACPPowerFlow(),
    GeneratorLimits(),
    BranchLimits(),
    VSCQuadraticLossNLP(converters = hvdc_vsc_converter_specs(hvdc_case)),
    GenerationCost(),
)
