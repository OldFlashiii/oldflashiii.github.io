Security Constraints and Cross-System Coupling#

Security and coupling components extend a base OPF into SCOPF, corrective OPF, network reconfiguration, cyber-physical optimization, or integrated energy-system optimization.

ContingencyConstraints#

Let \(\mathcal{K}\) be the contingency set and \(k=0\) the base case. Each scenario has its own state:

\[ x^k,\qquad k\in\{0\}\cup\mathcal{K} \]

Base-case constraints:

\[ g^0(x^0)=0,\qquad h^0(x^0)\le0 \]

Post-contingency constraints:

\[ g^k(x^k)=0,\qquad h^k(x^k)\le0,\qquad k\in\mathcal{K} \]

Preventive SCOPF shares controls:

\[ P_{g}^{k}=P_g^0 \]

Corrective SCOPF allows bounded recourse:

\[ |P_g^k-P_g^0|\le \Delta P_g^{corr} \]

CorrectiveRedispatch#

\[ P_{g,t}^{k}=P_{g,t}^{0}+\Delta P_{g,t}^{k} \]
\[ -R_g^{down,corr}\le \Delta P_{g,t}^{k}\le R_g^{up,corr} \]

Only selected units may participate:

\[ \Delta P_{g,t}^{k}=0,\qquad g\notin\mathcal{G}^{corr} \]

Corrective penalty:

\[ \min C(P^0)+\sum_{k\in\mathcal{K}}\pi_k \sum_g c_g^{corr}|\Delta P_g^k| \]

PTDF/LODF Screening#

\[ F_\ell = F_\ell^0 + \sum_i \mathrm{PTDF}_{\ell,i}\Delta p_i \]

Line outage approximation:

\[ F_\ell^{(m)}=F_\ell^0+\mathrm{LODF}_{\ell,m}F_m^0 \]

Thermal constraints:

\[ -F_\ell^{max}\le F_\ell^{(m)}\le F_\ell^{max} \]

This is useful for screening many contingencies before detailed AC SCOPF checks.

SwitchBigMFlow and Network Reconfiguration#

\[ z_{ij,t}\in\{0,1\} \]
\[ -M z_{ij,t}\le P_{ij,t}\le M z_{ij,t} \]

Closed-switch DC flow:

\[ -M(1-z_{ij,t}) \le P_{ij,t}-b_{ij}(\theta_i-\theta_j) \le M(1-z_{ij,t}) \]

Radial edge count:

\[ \sum_{(i,j)\in\mathcal{E}}z_{ij,t}=|\mathcal{N}|-1 \]

Connectivity can be enforced with single-commodity flow or spanning-tree constraints.

CommunicationDelayLimits#

\[ 0\le d_{\ell,t}^{cyber}\le \overline{d}_{\ell} \]
\[ D_{a,t}=\sum_{\ell\in path(a)}d_{\ell,t}^{cyber} \]
\[ D_{a,t}\le \overline{D}_a \]

MeasurementAvailability#

\[ m_{i,t}\in\{0,1\} \]
\[ |y_{i,t}^{meas}-y_{i,t}^{true}|\le M(1-m_{i,t})+\epsilon_i \]

ControlSignalRouting#

\[ \sum_{p\in\mathcal{P}_a} r_{a,p,t}=1,\qquad r_{a,p,t}\in\{0,1\} \]
\[ \sum_{a,p:\ell\in p} b_a r_{a,p,t}\le \overline{B}_{\ell} \]

MeasurementControlCoupling#

\[ u_{a,t}^{ctrl}\le m_{i,t}^{available} \]

or:

\[ |\Delta P_{g,t}^{ctrl}|\le \overline{\Delta P}_g u_{a,t}^{ctrl} \]

GasPowerCoupling#

Gas consumption of gas-fired units:

\[ q_{g,t}^{gas}=a_g P_{g,t}+b_g u_{g,t} \]

Gas nodal balance:

\[ \sum_{s\in\mathcal{S}_n}q_{s,t}^{supply} -\sum_{p\in\delta_n^+}q_{p,t}^{gas} +\sum_{p\in\delta_n^-}q_{p,t}^{gas} -\sum_{g\in\mathcal{G}_n^{gas}}q_{g,t}^{gas} =0 \]

Weymouth pipe equation:

\[ (q_{ij,t}^{gas})^2=K_{ij}\left(\pi_{i,t}^2-\pi_{j,t}^2\right) \]

SOCP or linear approximations can be used.

HeatPowerCoupling#

CHP feasible region:

\[ (P_{h,t},H_{h,t})\in\mathcal{F}_h^{CHP} \]

Linear approximation:

\[ \underline{P}_h u_{h,t}\le P_{h,t}\le \overline{P}_h u_{h,t} \]
\[ \underline{H}_h u_{h,t}\le H_{h,t}\le \overline{H}_h u_{h,t} \]
\[ a_h P_{h,t}+b_h H_{h,t}\le c_h u_{h,t} \]

References#

  • SCOPF formulation, challenges, and contingency filtering: Capitanescu et al., 2011.

  • Chance-constrained OPF and uncertainty-aware security constraints: Bienstock, Chertkov, and Harnett, 2014.

  • PTDF/LODF are standard DC network sensitivity tools; see Stott, Jardim, and Alsac, “DC Power Flow Revisited”, IEEE TPWRS, 2009.
