Network Formulations#

A network formulation describes which variables and equations represent the network.

Overview#

Formulation

Polar AC#

init_power!(ctx; formulation = :acp, state_variables = :polar)

Rectangular AC#

init_power!(ctx; formulation = :acr, state_variables = :rectangular)

IVR#

init_power!(ctx; formulation = :ivr, state_variables = :ivr)

W-Space SOCP#

init_power!(ctx; formulation = :wsoc, state_variables = :w_space)

The key relaxation is:

\[ (W_{ij}^r)^2 + (W_{ij}^i)^2 \le W_{ii} W_{jj}. \]

DC and PTDF#

init_power!(ctx; formulation = :dcp, state_variables = :dc)
init_power!(ctx; formulation = :ptdf)

AC/DC Hybrid#

init_power!(
    ctx;
    formulation = :acp,
    state_variables = :polar,
    network_type = :acdc,
    dc_state_variables = :voltage_square,
    hvdc_case = hvdc_case,
)
