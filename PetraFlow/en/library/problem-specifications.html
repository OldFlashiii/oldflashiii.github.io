Problem Specifications#

A problem specification describes what optimization or analysis problem is being solved. It is independent of the algebraic network formulation.

Supported and Emerging Specifications#

Specification

Typical components

Meaning

PF

solve_pf or a flow component

Solve steady-state voltages

ACOPF

AC flow + generator limits + branch limits + generation cost

AC optimal power flow

DCOPF

DCP/PTDF flow + generator limits + branch limits + generation cost

Linearized active-power OPF

Convex OPF Relaxation

W-SDP, W-SOC, QC, or branch-flow SOC + limits + cost

Convex relaxation of ACOPF

Storage Dispatch

Flow + storage + storage/generation cost

Multi-period storage scheduling

Unit Commitment

Flow + UC components + startup costs

Mixed-integer commitment

AC/DC Hybrid OPF

AC flow + DC grid + converter components

Hybrid AC/DC OPF

SCOPF

Base OPF + contingency and corrective components

Security-constrained OPF

Multi-Energy Optimization

Power + gas/heat networks + coupling

Integrated energy optimization

Cyber-Physical Optimization

Power + communication/measurement/control components

Security and resilience modeling

ACOPF Example#

init_power!(ctx; formulation = :acp, state_variables = :polar)
add_modules!(ctx, ACPPowerFlow(), GeneratorLimits(), BranchLimits(), GenerationCost())

The same problem can use a different formulation:

add_modules!(ctx, ACRPowerFlow(), GeneratorLimits(), BranchLimits(), GenerationCost())
add_modules!(ctx, IVRPowerFlow(), GeneratorLimits(), BranchLimits(), GenerationCost())

Convex OPF Relaxation Example#

W-space SDP relaxation:

init_power!(ctx; formulation = :w_sdp, state_variables = :w_sdp)
add_modules!(ctx, WSDPPowerFlow(), GeneratorLimits(), BranchLimits(), GenerationCost())

W-space SOCP or QC relaxation:

init_power!(ctx; formulation = :wsoc, state_variables = :w_space)
add_modules!(ctx, WSOCPowerFlow(), GeneratorLimits(), BranchLimits(), GenerationCost())

init_power!(ctx; formulation = :qc, state_variables = :w_space)
add_modules!(ctx, QCPowerFlowRelaxation(), GeneratorLimits(), BranchLimits(), GenerationCost())

AC/DC Hybrid Example#

init_power!(
    ctx;
    formulation = :acp,
    state_variables = :polar,
    network_type = :acdc,
    hvdc_case = hvdc_case,
)

Problem specifications are expressed by component composition; network formulations are expressed by init_power! and the selected power-flow component.
