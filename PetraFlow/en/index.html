PetraFlow Documentation#

PetraFlow is a Julia platform for power-system steady-state analysis, OPF modeling, modular optimization, and numerical solver research.

The documentation follows a PowerModels-style structure:

  • Manual: user workflows and tutorials.

  • Components: mathematical formulations, variables, parameters, solver classes, and references by component type.

  • Library: public API, problem specifications, network formulations, and component catalog.

  • Developer: extension guides for components, tests, and documentation.

Core Concepts#

PetraFlow separates:

  • Problem specifications: what problem is being solved, such as PF, OPF, DCOPF, SCOPF, storage dispatch, or AC/DC hybrid OPF.

  • Network formulations: which state variables and equations represent the network, such as polar, rectangular, IVR, W-space, branch-flow, or PTDF.

  • Modeling components: reusable JuMP writers for devices, limits, costs, security constraints, and coupling equations.
