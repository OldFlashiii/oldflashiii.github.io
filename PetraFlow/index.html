PetraFlow Documentation#

Power System Analysis · OPF · Modular Modeling

PetraFlow

A Julia platform for steady-state power-system analysis, optimal power flow, modular optimization modeling, and numerical solver research.

中文文档 English Docs Baseline GitHub

PetraFlow’s documentation is organized in a PowerModels-style structure:

  • Manual: user workflows and tutorials.

  • Components: component formulations, mathematical constraints, solver classes, and references.

  • Library: public API, problem specifications, network formulations, and component catalogs.

  • Developer: extension guides for components, tests, and documentation.
