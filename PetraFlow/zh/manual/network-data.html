网络数据#

PetraFlow 的电网数据分两层：

  • MATPOWERCase 和 HVDCMatACDCCase 保留原始 .m 文件的矩阵结构。

  • Network 是 PetraFlow 的 typed 数据结构，供潮流、稳态分析和优化建模复用。

读取 MATPOWER/PGLIB 算例#

net = load_network("case14")
net = load_network(joinpath(PETRAFLOW_DATA_DIR, "PGLIB", "pglib_opf_case118_ieee.m"))

在模块化优化中通常使用 PowerNetwork wrapper：

power = load_power_network("case14")

PGLIB-HVDC/MatACDC 数据：

case_path = joinpath(PETRAFLOW_DATA_DIR, "PGLIB-HVDC", "case5_3_he.m")
power = load_power_network(case_path)
hvdc_case = load_pglib_hvdc_case(case_path)

变压器和无功补偿#

PetraFlow 使用普通 Branch 表示线路和变压器。若 tap 非 0/1 或 shift 非 0，则该支路具有变压器或移相器含义。

Ybus 构造会考虑：

  • series admittance；

  • line charging；

  • bus shunt；

  • off-nominal tap；

  • phase shift。

用户构造网络时可以直接使用：

add_transformer!(b, 1, 2; r = 0.01, x = 0.05, tap = 1.05, shift = 3.0)
add_shunt!(b, 2; gs = 0.0, bs = 2.0)
add_capacitor!(b, 2; q_mvar = 5.0)
add_reactor!(b, 2; q_mvar = 1.0)

add_shunt! 默认累加已有 shunt，传入 replace = true 可以替换。

常用查询#

nbus(net)
ngen(net)
nbranch(net)
nstorage(net)

bus_by_id(net, 14)
gen_by_id(net, 1)
branch_by_id(net, 1)

total_load(net)
total_generation(net)
total_shunt(net)
count_transformers(net)
count_active_branches(net)

矩阵构造#

mats = make_ybus(net)
bdc = make_bdc(net)
Bp, Bpp = makeB(net)

make_ybus 返回 Ybus/Yf/Yt 相关矩阵；make_bdc 和 makeB 面向 DC 潮流和 fast-decoupled 潮流。
