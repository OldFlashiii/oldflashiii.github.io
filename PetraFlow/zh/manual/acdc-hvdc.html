AC/DC 与 HVDC#

PetraFlow 默认将电力网络视为纯 AC 网络。只有显式设置 network_type = :acdc 或 :hybrid 时，才会初始化 DC 侧变量。

读取 PGLIB-HVDC#

case_path = joinpath(PETRAFLOW_DATA_DIR, "PGLIB-HVDC", "case5_3_he.m")
power = load_power_network(case_path)
hvdc_case = load_pglib_hvdc_case(case_path)

HVDCMatACDCCase 保存：

  • AC bus/gen/branch/gencost；

  • DC bus；

  • DC converter；

  • DC branch；

  • dcpol；

  • 原始字段名称。

初始化混联系统#

ctx = power_context(power; T = 1)

init_power!(
    ctx;
    formulation = :acp,
    state_variables = :polar,
    network_type = :acdc,
    hvdc_case = hvdc_case,
)

初始化后会记录：

ctx.metadata[:power_network_type] == :acdc
ctx.metadata[:power_is_hybrid_acdc] == true

并根据 DC formulation 创建变量，例如：

  • dc_bus_injection；

  • dc_voltage；

  • dc_voltage_square；

  • dc_line_flow；

  • dc_line_current_square。

DC 侧参数#

参数

说明

默认 DC 侧电压平方建模

根据 formulation 自动创建变量

dc_state_variables = :voltage

创建 dc_voltage

dc_state_variables = :voltage_square

创建 dc_voltage_square

dc_bus_count

手动指定 DC 母线数量

dc_line_count

手动指定 DC 支路数量

dc_lines

手动传入 DCLineSpec

HVDC 辅助规格#

dc_lines = hvdc_dc_line_specs(hvdc_case)
vsc = hvdc_vsc_converter_specs(hvdc_case)
droop = hvdc_droop_specs(hvdc_case)
lcc = hvdc_lcc_converter_specs(hvdc_case)

这些 helper 将 MatACDC 矩阵转换为 PetraFlow 组件可直接使用的 spec。
