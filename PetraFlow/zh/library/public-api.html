公共 API#

本页汇总 PetraFlow 当前推荐的公共 API。更底层的内部函数可以从源码和测试中查找，但用户文档优先使用这里列出的入口。

数据读取#

函数

作用

load_matpower_case(path)

读取 MATPOWER .m 文件，返回矩阵式 MATPOWERCase

load_network(name_or_path)

读取并转换为 typed Network

load_power_network(name_or_path)

读取并包装为模块化建模使用的 PowerNetwork

load_hvdc_matacdc_case(path)

读取 PGLIB-HVDC/MatACDC 文件

load_pglib_hvdc_case(name_or_path)

从 data/PGLIB-HVDC 或指定路径读取 HVDC case

list_cases

列出可用算例

resolve_case_path

解析算例路径

网络构造#

函数

作用

build_network

从 typed 元件直接构造 Network

build_power_network

用户友好的 builder API，返回 PowerNetwork

add_bus!

添加母线

add_generator!

添加发电机

add_branch!

添加线路或普通支路

add_transformer! / add_transformer_branch!

添加变压器支路

add_shunt!

添加或替换 bus shunt

add_capacitor!

添加电容器无功补偿

add_reactor!

添加电抗器无功补偿

add_storage!

添加储能元件

add_gencost!

添加发电成本

OPF 与模块化建模#

函数

作用

solve_opf, runopf

应用层 OPF 入口

power_context, modular_context

创建模块化建模上下文

init_power!

初始化电力网络变量

add_modules!, add_component!

添加组件

build_modular_model!

构建 JuMP 模型

solve_modular_model!

配置求解器并求解

recover_optimization_result

恢复模块化模型结果

available_solution_variables

查看可查询的模块化变量

solution_variables

返回模块化变量结果字典

solution_value / variable_solution

查询变量矩阵、切片或标量

bus_solution

查询母线电压、负荷、聚合出力等结果

generator_solution

查询机组出力、启停状态和上下限

branch_solution

查询支路潮流、视在功率、loading 和相角差

storage_solution

查询储能 SOC、充放电和能量

solution_status

查询终止状态、目标值、问题类型和求解器

modular_solution_summary

查询状态、变量列表和组件摘要

power_solution

打包查询电力系统结果表

registered_optimization_modules

已注册组件目录

find_optimization_modules

按关键词查找组件

infer_modular_problem_class

推断 LP/QP/NLP/SOCP/MILP 等问题类型

select_optimizer

选择外部求解器
