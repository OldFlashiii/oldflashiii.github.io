网络 Formulations#

网络 formulation 描述“用什么变量和方程表示网络”。PetraFlow 当前支持 AC、DC、W-space、branch-flow 和 AC/DC hybrid 相关变量空间。

AC Polar#

变量：

init_power!(ctx; formulation = :acp, state_variables = :polar)

主要变量：

  • power.V

  • power.theta

  • power.Pg, power.Qg

  • power.Pbr_from, power.Qbr_from, power.Pbr_to, power.Qbr_to

AC Rectangular#

init_power!(ctx; formulation = :acr, state_variables = :rectangular)

主要变量：

  • power.Vr

  • power.Vi

  • 发电机功率；

  • 支路功率。

IVR#

init_power!(ctx; formulation = :ivr, state_variables = :ivr)

主要变量：

  • power.Vr, power.Vi

  • power.Ir_from, power.Ii_from, power.Ir_to, power.Ii_to

  • power.Pbr_from, power.Qbr_from, power.Pbr_to, power.Qbr_to

W-space SOCP#

init_power!(ctx; formulation = :wsoc, state_variables = :w_space)

主要变量：

  • power.Wii

  • power.Wr

  • power.Wi

核心松弛：

\[ W_{ij}^r{}^2 + W_{ij}^i{}^2 \le W_{ii} W_{jj} \]

W-space SDP#

init_power!(ctx; formulation = :w_sdp, state_variables = :w_sdp)

主要变量：

  • power.W_sdp[t]：实数提升的半正定矩阵；

  • power.Pbr_from, power.Qbr_from, power.Pbr_to, power.Qbr_to。

该 formulation 保留完整电压外积矩阵的凸松弛，适合验证 SDP 下界和小规模凸松弛算例。

QC relaxation#

init_power!(ctx; formulation = :qc, state_variables = :w_space)

主要变量沿用 W-space SOCP 的 Wii、Wr、Wi，并通过凸包络强化电压乘积和非线性项。

DC 和 PTDF#

角度 DC：

init_power!(ctx; formulation = :dcp, state_variables = :dc)

PTDF：

init_power!(ctx; formulation = :ptdf)

PTDF 形式适合安全约束和快速线性调度。

AC/DC Hybrid#

init_power!(
    ctx;
    formulation = :acp,
    state_variables = :polar,
    network_type = :acdc,
    dc_state_variables = :voltage_square,
    hvdc_case = hvdc_case,
)

DC 侧可创建：

  • dc_bus_injection

  • dc_voltage

  • dc_voltage_square

  • dc_line_flow

  • dc_line_current_square
